PastePaw is a lightweight, privacy-focused clipboard manager for macOS. Built with Rust for performance and designed to feel native.

Download for macOS Star on GitHub

Everything you need.

Private by Design

All your clipboard data is stored locally in a SQLite database. No tracking, no cloud sync, no data leaves your machine.

Blazing Fast

Built with Rust and Tauri. Minimal memory footprint and instant response times.

AI Assistant

Summarize, translate, and fix grammar directly from your clipboard using your own API keys.

Instant Search

Find anything you've copied in seconds. Powerful filtering and fuzzy search help you navigate your history effortlessly.

Smart Folders

Organize frequent clips into custom folders for quick access later.

Sensitive Filtering

Automatically ignores content from password managers and sensitive applications.

Native Look

A beautiful interface that follows macOS system themes and design guidelines.
