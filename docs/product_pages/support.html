Frequently Asked Questions

Q: How do I open PastePaw?
Press ‚åÉ‚áßV (Control + Shift + V) on your keyboard. You can customize this hotkey in the app's Settings.
Q: Is my clipboard data secure?
Yes! All your clipboard data is stored locally on your device in an encrypted SQLite database. Nothing is sent to any server or cloud service. Read our Privacy Policy for more details.
Q: Can I exclude certain apps from being recorded?
Yes! Go to Settings ‚Üí Ignored Applications and add any apps you want to exclude (like password managers or banking apps). PastePaw will not record clipboard content from these apps.
Q: How do I delete my clipboard history?
You can delete individual items by selecting them and pressing Delete, or clear all history from Settings ‚Üí Clear All History.
Q: Does PastePaw work on Apple Silicon (M1/M2/M3) Macs?
Yes! PastePaw is built as a Universal Binary and runs natively on both Intel and Apple Silicon Macs.
Q: How do I use the AI features?
To use AI features (summarize, translate, explain code, fix grammar), you need to provide your own API key from an AI provider (OpenAI, DeepSeek, or compatible services). Configure this in Settings ‚Üí AI Settings. Right-click any clipboard item to access AI actions.
Q: Why does PastePaw need Accessibility permissions?
Accessibility permissions are only needed if you want to use the auto-paste feature. This allows PastePaw to simulate keyboard input (Cmd+V) to automatically paste items. You can use PastePaw without this permission and paste items manually.
Q: How do I uninstall PastePaw?
Simply drag PastePaw from your Applications folder to the Trash. All clipboard data will be removed when you delete the app. If you want to completely remove all preferences, also delete ~/Library/Application Support/me.xueshi.pastepaw.
Q: Can I sync my clipboard history across multiple Macs?
Currently, PastePaw stores all data locally and does not support cloud sync. This is by design to ensure maximum privacy and security. Sync features may be considered for future versions.
Q: Is PastePaw free?
Yes! PastePaw is free and open source software licensed under GPL-3.0. You can view and contribute to the source code on GitHub.

System Requirements

  • macOS Version: macOS 11.0 (Big Sur) or later
  • Architecture: Intel and Apple Silicon (Universal Binary)
  • Disk Space: ~20 MB for the app, plus storage for clipboard history
  • Memory: Minimal RAM usage (~50-100 MB)

Getting Started

  1. Download PastePaw from the latest release
  2. Open the downloaded DMG file and drag PastePaw to your Applications folder
  3. Launch PastePaw from your Applications folder
  4. Grant Accessibility permissions if you want to use auto-paste (optional)
  5. Press ‚åÉ‚áßV to open your clipboard history
  6. Start copying and enjoy unlimited clipboard history!
